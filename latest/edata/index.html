Edge Metadata

Edge Metadata

Graft supports the assignment of metadata to vertices, and adopts a tabular approach to storage. Edge metadata is stored as an AbstractDataFrame. The adjacency matrix stores the row number for each edge, i.e. it serves as an edge to index map, or an index table to the edge metadata table.

The following example demonstrates edge metadata:

julia> using Graft

julia> g = completegraph(10);

julia> setlabel!(g, map(string, 1 : 10));

julia> eit = edges(g);

julia> # Create a new edge property
       seteprop!(g, :, 1 : 90, :p1);

julia> # Fetch an entire column from the vertex table
       geteprop(g, :, :p1)
